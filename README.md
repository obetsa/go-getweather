## go-getweather  -  simple weather application for getting your current forecast

- Golang / Gin
- Java Script / Axios
- Bootstrap 5
- API by www.weatherapi.com

## Docker run
sudo docker run -d --restart always -p 8080:8080 obetsa/go-getweather:latest

## Terraform
terraform init
terraform apply

## Kubernetes
kubectl apply -f .
